Anagram

Anagram is shuffling around the letters in a word or phrase using all the original letters exactly once.

Example: A Gentleman => Elegant Man

Example: Silent => Listen

Pros:

  • Easy to create if you just want to randomly shuffle
  • Can be fun for the solver if message is a fun phrase

Cons:

  • Can easily use an anagram solver online to find each word
  • Easy to recognize word if word is short
  • Hard to make if you want to be witty and make a message using real words

About Giving Hints:

  • May not be needed since anagrams are generally recognizable, but you can give if you want to
  • Solvers may take awhile to decode long words, so it is advised to give them the first and last letter of the original word
