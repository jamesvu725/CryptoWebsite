Binary Code

Binary code is a code that uses two symbols to represent data. 0s and 1s are most commonly used. For this cryptogram, we can either start counting the alphabet at 1 or use the ASCII table values and start at 65 for A or 97 for a.

Example: A(1) = 00001 or A(65) = 01000001 or a(97) = 01100001

Pros:

  • Easy to solve once you figure it out

Cons:

  • Will be hard to figure out if you are not familiar with it

About Giving Hints:

  • Giving A = 00000001 is a good hint since it lets the solver know the starting point, starting in the middle will make it harder if it is the only hint
  • It is recommended to use binary code starting with 1 instead of 65 or 97 since most people are not familiar with ASCII code
  • The last 5 digits in ASCII is exactly the same as counting starting from 1, so you can actually ignore the first 3 digits.
