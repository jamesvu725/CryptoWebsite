Count the Circles

This is a simple cryptogram where you count the number of circles in each number and the number of circles corresponds to a letter.

Example:

The number 0 has 1 circle in it, so the letter is A.

The number 8 has 2 circles in it, so the letter is B.

Pros:

  • Easy to solve once you figure it out

Cons:

  • May take time to count each line of circles
  • Not practical spacewise since each letter takes up a lot of space

About Giving Hints:

  • Give at least one hint showing a number and its letter counterpart. Example: 0689 = E
  • If they are stuck, tell them to not overthink it
  • If they are really stuck, tell them counting is essential

Use the Letter to Number tool to find how many circles you need for each letter. Create your message by adding numbers to reach your circle number. Note: 4 does not have a circle since the number 4 can be written with or without the hole.
