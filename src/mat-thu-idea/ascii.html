ASCII

ASCII stands for American Standard Code for Information Interchange. ASCII code is the numerical representation of a character. For this cryptogram, every letter is substituted with its ASCII code.

Example: A = 65, a = 97

Pros:

  • The solver doesn't need to know the ASCII table to solve it if they are given hints as a reference

Cons:

  • May be difficult to understand

About Giving Hints:

  • Give at least 1 hint as a reference to know where the alphabet is
  • Give at least 2 hints as reference if both lowercase and uppercase is used in the solution
  • Give A = 65 or a = 97 if you want to make it easier for the solver
  • Give a random letter in the middle of the alphabet if you want it to be harder for the solver
